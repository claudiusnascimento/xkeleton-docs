Instalation (without Docker)

Clone the repo#

git clone https://github.com/claudiusnascimento/xkeleton.git

Enter in project folder

cd xkeleton

Delete the followed docker files and folders (not madatory)#

sudo rm -rf docker
rm Dockerfile docker-compose.yml

Install project#

composer install

Folder Permissions#

sudo docker-compose exec app chmod -R 777 storage/
sudo docker-compose exec app chmod -R 777 bootstrap/

Copy .env.axample#

cp .env.example .env

Generate the key#

php artisan key:generate

Database#

Set correct the database options in .env file for your environment.

This step is note necessary if you install with docker

Run migrations#

php artisan migrate

Seeding#

Open the file database/seeds/DatabaseSeeder.php and uncomment the seeds that you want and...

php artisan db:seed
After that is good practice comment the seeds again

Access the admin#

http://localhost:8080/admin

Log with credentials#

  • email: admin@admin.com
  • pass: 123456
